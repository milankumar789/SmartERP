# SmartERP
SmartERP – A full-stack ERP system built with Spring Boot, React, and PostgreSQL featuring authentication, inventory management, customer &amp; supplier management, sales, purchases, reporting, and keyboard-driven navigation.

Features
✔ JWT Authentication

✔ Dashboard

✔ Customer Management

✔ Supplier Management

✔ Category Management

✔ Product Management

✔ Purchase Management

✔ Sales Management

✔ Reports

✔ Inventory Tracking

✔ Keyboard Shortcuts

✔ Pagination

✔ Search

✔ Responsive UI
